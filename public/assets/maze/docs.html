1. Game Interface:

The interface is divided into main areas to help you interact and program.

  1. Maze Area: Where your character will move to complete the challenge.
  2. Toolbox: Contains all the command blocks you can use in the current level.
  3. Workspace: Drag and drop command blocks here to build your program.
  4. Control Area: Contains buttons to run the program, replay, and view instructions.
  5. Title Bar: Displays the game name, levels, and allows you to change the language.

2. Scoring:

For each level, you can earn a maximum of 3 stars. The number of stars depends on the number of blocks you use compared to the optimal solution.

  • 3 stars : Excellent solution, using the fewest possible blocks.
  • 2 stars : Good solution, completed the level.
  • 1 star : Completed the level, but it could be much more optimized.

3. Working with Blocks:

Drag blocks from the toolbox and drop them into the workspace to connect them.

Right-click on a block to open the context menu:

  • Duplicate Block: Select "Duplicate" to create a copy of the block (and any blocks inside it).
  • Delete Block: Select "Delete Block" or drag the block to the trash can icon.

4. Basic Blocks:

"move forward" block

Description: This command tells the character to move one step forward in the direction it is facing.

Usage: Use this block whenever you want the character to advance in the maze.

"turn" block

Description: This command rotates the character 90 degrees on the spot. You can choose to turn left or right.

Usage: When the character needs to change direction, for example, at a turn or when facing a wall.

"repeat until" block

Description: Any commands placed inside this block will be executed repeatedly until the character reaches the finish goal.

Usage: Useful for moving along a long path without manually repeating blocks.

"if path ... do" block

Description: This conditional block allows the character to "check" its surroundings. The commands inside are only executed if there is a path in the chosen direction.

Usage: Helps the character avoid bumping into walls and allows it to navigate turns.

"repeat ... times" block

Description: Executes the commands inside a specified number of times.

Usage: Very useful when you know exactly how many times an action needs to be performed (e.g., moving across a straight path of 5 squares).

"if path ... do ... else" block

Description: This is an extension of the "if" block. It checks a condition: if true, it executes the first set of commands (in the "do" section); if false, it executes the second set of commands (in the "else" section).

Usage: Very useful when you want the character to have two clear action choices at a decision point. For example: "if there is a path ahead, move forward, otherwise, turn left".
